# Marchel Shevchenko — AI Architect & Builder Portfolio

Welcome to the official portfolio repository of **Marchel Shevchenko**, an AI Architect specializing in Multimodal AI, Foundation Models, and Indonesian NLP. This portfolio is not just a static website; it is an interactive, terminal-driven, and highly immersive experience designed to showcase advanced AI projects, interactive games, and a comprehensive professional profile.

## 🌟 Key Features

This portfolio is built with a unique blend of retro-futuristic aesthetics and modern web technologies, offering a rich set of features:

*   **Interactive Terminal Interface:** A fully functional, globally accessible terminal overlay (toggle with `Ctrl+\`` or the `> TERMINAL_` button). It supports commands like `help`, `about`, `projects`, `skills`, `sudo`, and even hidden easter eggs.
*   **Immersive Games & Experiences:** A dedicated "GAME" menu featuring interactive web-based experiences:
    *   📖 **Origin Story:** An interactive narrative of Marchel's journey.
    *   🤚 **Gesture CV:** A computer vision-powered interactive CV.
    *   ⚔️ **Nusantara RPG:** A text-based RPG adventure.
    *   🧠 **Neural Builder:** A drag-and-drop neural network architecture game.
    *   🗺️ **404 Adventure:** A creative take on the standard 404 error page.
    *   ♟️ **Chess vs AI:** Play chess against an AI directly in the browser.
    *   🎮 **MOBA (Aksara Kingdom):** A web-based MOBA concept game.
*   **Dynamic Backgrounds & Visuals:** Features semantic language particle networks, CRT scanline effects, and WebGL-powered interactive backgrounds (like the Wormhole effect).
*   **Bilingual Support:** The terminal and several components support switching between English and Bahasa Indonesia.
*   **Admin CMS:** A built-in, secure Admin dashboard (`/admin`) for managing portfolio content dynamically via Supabase.
*   **Security Measures:** Built-in scripts to disable right-click, text selection, and developer tools to protect the interactive experience.

## 🛠️ Technology Stack

*   **Frontend:** HTML5, CSS3 (Custom Variables, Grid/Flexbox), Vanilla JavaScript (ES6+).
*   **Visuals & Rendering:** HTML5 Canvas API, WebGL, Chart.js (for data visualization).
*   **Backend/Database (Admin):** Supabase (PostgreSQL, Authentication).
*   **Deployment:** Vercel (with custom `vercel.json` for clean routing and caching headers).
*   **Fonts:** Google Fonts (VT323, Share Tech Mono, Press Start 2P, Orbitron, Rajdhani).

## 🚀 Local Development

To run this portfolio locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/chenko-the-sorcerers/marchel-shevchenko.git
    ```
2.  Navigate to the directory:
    ```bash
    cd marchel-shevchenko
    ```
3.  Serve the files using a local web server. For example, using Python:
    ```bash
    python3 -m http.server 8000
    ```
4.  Open your browser and visit `http://localhost:8000`.

*Note: Because the project uses ES6 modules and fetch API for some components, opening the HTML files directly via `file://` protocol may cause CORS errors. Always use a local server.*
